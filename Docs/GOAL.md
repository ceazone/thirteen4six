# GOAL.md — Airbnb Command Center

## What is this product?
A single-page host command center that consolidates all Airbnb business data, pricing intelligence, and operational tools into one local dashboard — replacing the need to jump between Airbnb, PriceLabs, spreadsheets, and markdown files.

## Who is it for?
Cesar Santos — solo host running a 1-bed/1-bath basement suite in Columbia Heights, DC ("Private DC Suite · Free Parking · 9 Min to Metro"). Currently earning ~$1,900/month, targeting $3,800/month.

## What does success look like?
- Open one HTML file and see: current month revenue, occupancy, ADR, upcoming bookings, pricing alerts, and progress toward $3,800 target
- Pricing intelligence: PriceLabs data, event calendar, seasonal recommendations — all visible without logging into another tool
- Operational checklists: what needs doing this week, what's overdue, what's coming up
- Guest tools: message templates, review response drafts, check-in instructions
- Historical tracking: month-over-month revenue trends, booking rate trends, ADR trends
- Works offline — no server, no database, just a local HTML file with embedded data

## Guiding Principles
1. **One file, one truth.** The command center is the single place to check. If it's not here, it doesn't exist.
2. **Data over decoration.** Every pixel should inform a decision. No filler sections.
3. **Update-friendly.** Designed so monthly data refreshes are a 5-minute copy-paste, not a rebuild.
4. **Host-first.** Every feature must serve the daily workflow of running this one listing.
5. **Progressive build.** Ship a working version each sprint. It's always usable, just gets better.
